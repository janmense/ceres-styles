# Ceres Colorful – Ceres Plugin in Colorful-Style

<div class="container-toc"></div>

## Activate Colorful Theme

Go to plentymarkets Backend Plugin Area. You have to provide the Colorful theme in productive and assign the client (store). Switch to "Content" tab and check the "Template: Style" entry in "CeresColorful" section. Save, finished!

<div class="alert alert-info" role="alert">
    Ceres and IO Plugin are required.
</div>
